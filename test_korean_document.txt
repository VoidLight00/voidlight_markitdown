# VoidLight MarkItDown 테스트 문서

안녕하세요! 이것은 **VoidLight MarkItDown**의 한국어 처리 기능을 테스트하는 문서입니다.

## 주요 기능

1. **한글 텍스트 처리**: 완벽한 한글 인코딩 지원
2. **다양한 형식 변환**: TXT, HTML, PDF, DOCX 등
3. **한국어 NLP 통합**: KoNLPy, Kiwipiepy 등 활용

## 테스트 내용

### 한글과 영어 혼용
Hello World! 안녕하세요! 이렇게 한글과 English를 섞어서 사용해도 문제없습니다.

### 특수문자와 이모지
한국어 문장에서 사용하는 특수문자들: 「」『』【】〔〕
이모지도 지원합니다: 😊 🎉 🚀

### 코드 블록
```python
def 인사하기(이름):
    print(f"안녕하세요, {이름}님!")
    
인사하기("VoidLight")
```

## 마무리

VoidLight MarkItDown은 Microsoft의 MarkItDown을 기반으로 한국어 처리 기능을 강화한 버전입니다.
더 나은 한국어 문서 처리를 위해 계속 발전하고 있습니다! 🇰🇷
